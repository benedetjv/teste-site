# Hérnia de Disco x Protusão Discal

Ao receber o laudo de uma ressonância magnética, muitos pacientes se assustam com termos como "abaulamento", "protusão" e "hérnia". Vamos esclarecer o que cada um significa.

## A Anatomia do Disco

O disco intervertebral é como um "amortecedor" entre as vértebras. Ele tem uma capa dura externa (**ânulo fibroso**) e um recheio gelatinoso (**núcleo pulposo**).

## 1. Abaulamento e Protusão Discal
São o início do problema.
- O disco perde altura e se "esparrama" um pouco para os lados.
- A capa externa (ânulo) **ainda está intacta**, mas deformada.
- Pode causar dor se encostar em algum nervo, mas muitas vezes é assintomática ou causa apenas dor local nas costas ("travamento").

## 2. Hérnia de Disco (Extrusão)
É a fase avançada.
- A capa externa **se rompe**.
- O gel interno vaza (extravasa) para dentro do canal vertebral.
- Esse gel inflama violentamente os nervos, causando a **dor ciática** (dor que desce para a perna) ou **cervicobraquialgia** (dor que desce para o braço).

## Qual é mais grave?

A **Hérnia Extrusa** é clinicamente mais agressiva, causando dores incapacitantes e perda de força. Porém, curiosamente, o corpo tem mais facilidade em "reabsorver" (secar) uma hérnia grande do que uma protusão pequena, pois o material vazado é rapidamente atacado pelo sistema imune.

## Tratamento É Cirúrgico?

Para ambos os casos, **90% dos tratamentos NÃO são cirúrgicos**.
Utilizamos:
- Medicação específica para dor neuropática.
- Fisioterapia especializada.
- **Infiltrações na coluna** (Bloqueios) para lavar a inflamação e acelerar a cura.

A cirurgia (endoscopia de coluna) fica reservada apenas para falhas no tratamento conservador ou casos de perda de força progressiva.
